this is the tutorial to write shell file and execute them in the terminal

1. create a file with .sh extension

2. enter the following commands on the top
 #!/bin/bash


3. Write the commands u would normally write in a terminal

4. save the file and exit

5. change the file to execute mode by typing
    chmod +x <filename>

6. now run the file by typing
  ./<name of the file>
