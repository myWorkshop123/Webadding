1. Require the module
2. create the url which is ("mongodb://localhost:27017" + "<dbName>")
3. mongoose.connect()
    this function is used to connect with the database
    mongoose.connect(url)

4. Create the schema // like the create table in sql
    const schema = new mongoose.Schema({})

5. Create the model/ collection
    const <name of the var> = mongoose.model('<name of the collection>', schemaVariable )

6. Create the document
     create a new object from the model specified above and fill the values following
     the schema in the dictionary manner
     	const fruit = new <name of the model> (
		{  name:'' .......etc }
		);
7. Insert the document
      insert the document in the database using the
	documentName.save() command

			   );
