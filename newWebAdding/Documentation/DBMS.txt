
insert into table(column-1, column-2 .... column-n) values(value-1 ,
value-2 ,........ value-n);
	this command is used to give only some values to the columns
	inside the table without having to fill all the columns

	for eg:--
	    if we used insert into table values(val-1 , val-2
	    .... val-m);
	    then the table will insert all the values inside all the
	    columns
