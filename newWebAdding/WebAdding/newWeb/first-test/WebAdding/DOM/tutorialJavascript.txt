ADD JS IN HTML 



to incorporate js in html there are three ways
 1. Inline js
      this is added inside the body tag by typing onload
      for eg
        <body onload="alert('hello');">

            <h1>Hello World </h1>


        </body>


 2. Internal js
       this is added inside the body section of the html
       for eg

       <body>

            <script>
              alert("hello");
            </script>


        </body>

 3. Externel js
      this is added as a linked javascript file in the html
      it is added  in this way
      <script type="text/javascript" src="index.js">

      it is added at the end of the body tag
        because if it is going to change an html element then it should know about that html
        element,
        if it is added in the head tag then it will not be knowing the html element/tag
        thus it will show an error

        So,
          the best practice is to add it in the end of the body tag

ADDING CSS CLASS 



We can also add a css class to the element in the console if the css class is already added but 
is not attached with the css element 

First Select the element by using the getElement or querySelector method 

document.querySelector("h1").classList.add(css Class Name);

there are three types of functions used in classList Object
1. classList.add()
    this will add the css class to the element 

2. classList.remove()
    this will remove the css class 

2. classList.toggle()
    this will toggle the css class i.e it will remove the class if added and vice versa
